快乐之道 - 什么是游戏？

Mingo's Blog
2017-08-28
快乐之道, 读书

游戏乐趣

游戏可能是现实世界的抽象，也可能是用更形象的方式表达现实世界中的抽象。它更很容易被大脑识别，识别成一种或者多种模式，这取决取游戏的内容。

我们玩游戏基本的想法是从游戏取得乐趣，实际上，这个想法如果解析到更深层的话，那么取得乐趣实际上是为了刺激了我们的大脑，而除了游戏，还有许多行为可以刺激到大脑，快乐是一切让我们的大脑感觉愉悦的东西，它使得体内的安多酚释放。

我们可通过很多行为让安多酚释放：

  • 不同品种的鸡尾酒
  • 美妙的音乐
  • 阅读书籍
  • 服用可卡因
  • 吃好吃的食物

这些都会产生同一类化学物质，我们的大脑其实一直在服用着各种药物。其中有很大一方面是由于我们学会了某种本领、掌握了某项技能、得到了某类体验等等，而触发了美妙的感觉。

游戏通过提供一种学习体验来获得快乐，在玩游戏时，学习游戏中的规则让我们兴奋，而相反，当一个游戏不能教给你更多东西时就会带来厌烦感。井字游戏就是一个很好的例子，玩家很快地学会游戏规则，并且迅速地掌握了它，在这个过程中得到了快乐，但当玩家熟悉了整个游戏的变化，甚至达到每一盘都是平局或者赢的情况时，整个游戏就没有了乐趣，因为再也没有可以学习的地方了。

当然当你在和你的孩子、女朋友和朋友玩井字游戏时仍然能够从中得到快乐，这不是因为游戏本身提供的，而是游戏外的变量附加到游戏上了，整个游戏已经变成一个社交工具。这可能就是我从书中得到的“游戏应该如何设计”的初始概念。

厌倦感从何而来 （摘抄）

  • 玩家通过最初5分钟就弄清了游戏的玩法，从而觉得这样的游戏没什么意思，就像成年人玩井字棋游戏一样。这并不意味着玩家真的完成了游戏，他可能只是找到一种足够好的战略或得到某种启发。“太简单了”，这很可能是玩家对这种游戏的评价。
  • 玩家发现游戏要达到可能的变化需要很多东西，但是这些东西却无法令他们提起兴趣。这就好像说：“是的，棒球其实有一大堆东西，但是记住那些20年来的RBI打点统计对我来说没有任何用处。”
  • 除了噪音就没其他东西了，玩家完全发现不了任何模式，“这太难了”。
  • 游戏中的模式变化得太慢了，这会使游戏被过早地认为没什么可玩的而被放弃，即使它确实有深度。“难度出现得太慢了”。
  • 游戏中的模式变化得太快了，这会使玩家因控制不了而放弃，从而使游戏又再一次成为噪音。“太难了，太快了。”玩家会这样说。
  • 玩家也许精通游戏中的所有模式，这样愉悦感就会消失殆尽，“我完全打败它了”。

上述任何一点都会使玩家感到厌倦。实际上，有些会令人产生厌倦感加挫败感，而有些则是产生厌倦感加胜利感，等等。如果你的目的是为了快乐（也就是“让玩家保持学习状态”），那么厌倦感就是那个告诉你已经失败了的信号。

游戏是什么

理论情况下，一个好的游戏，在于合理的安排内容，设定适合的参数，以让玩家永远有无限玩下去的兴趣。“无限”是不太可能做到的，所以书中的定义显得更真实：

一个好游戏的定义就是“能够在玩家停止游戏前，把应该提供的每一样东西都教给玩家”
